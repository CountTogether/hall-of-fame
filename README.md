# Hall of Fame – CountTogether Security Researchers

This repository serves as the data source for the [CountTogether Security Hall of Fame](https://counttogether.app/security/hall-of-fame) — a public page honoring individuals who have responsibly disclosed security vulnerabilities in CountTogether.

## 🙏 Thank You

We deeply appreciate the efforts of security researchers who help make CountTogether safer for everyone. If you've discovered a vulnerability and reported it responsibly, you may be listed here as a token of our gratitude.

## Report a Vulnerability

Found a security issue? Please report it to us via email:

**[security@counttogether.app](mailto:security@counttogether.app)**

Please include as much detail as possible:
- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Any relevant links, screenshots, or proof-of-concept

We aim to respond within **72 hours** and will keep you informed throughout the process.

## Responsible Disclosure Policy

Please review our full [Security Policy](https://counttogether.app/security) before submitting a report.
